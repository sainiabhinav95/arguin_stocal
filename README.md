Jupyter Notebooks for the exercises in the book:
A First Course in Stochastic Calculus
Louis-Pierre Arguin

https://bookstore.ams.org/amstext-53/




## How to use this repo:
1. Install UV
   - https://docs.astral.sh/uv/getting-started/installation/
2. Clone the repo
   ```bash
git clone https://github.com/sainiabhinav95/arguin_stocal.git
```
3. Sync the repository
   ```bash
uv sync
```
4. Run the Jupyter server
   ```bash
uv run --with jupyter jupyter lab
```
5. Open the `src/` folder to view the notebooks.
